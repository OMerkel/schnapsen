# German Cards with German Suits Doublefaced

Copyright 2026 Oliver Merkel, <merkel.oliver@web.de>

## Structure

52-card German deck with these four suits:

- Eichel
- Blatt
- Herz
- Schellen

Each suit contains these ranks:

- Ass (Ace)
- 10
- König
- Ober
- Unter
- 9
- 8
- 7
- 6
- 5
- 4
- 3
- 2

## Assets

Run the generator to create the complete SVG deck in this folder:

```bash
python generate_cards_german_double.py
```

Or run the VS Code task `Generate Cards German Double Deck`.

Generated files:

- `eichel_ass.svg` through `eichel_2.svg`
- `blatt_ass.svg` through `blatt_2.svg`
- `herz_ass.svg` through `herz_2.svg`
- `schellen_ass.svg` through `schellen_2.svg`
- `carte_german_double_back.svg`
- `carte_german_double_sheet.svg`

The card faces are rendered as traditional-style vector artwork intended to stay
close to a classic printed German deck look while remaining lightweight and
editable.
